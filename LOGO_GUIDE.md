# BuyOrBye Logo Guide

This guide is meant for image-generation tools like Gemini so you can quickly explore multiple logo directions for **BuyOrBye**.

## Brand Summary

**Brand name:** BuyOrBye

**What the product does:**  
A web app that helps people compare products by price, quality, and value before they buy.

**Brand personality:**  
Smart, decisive, trustworthy, modern, clear, slightly bold.

**Core idea:**  
The brand should feel like a product decision engine, not a discount coupon site.

## Creative Direction

The logo should communicate:

- product comparison
- smart decision-making
- buy vs skip
- confidence before purchase

The logo should avoid feeling:

- cheap
- noisy
- gimmicky
- overly cartoonish
- too generic like a random ecommerce brand

## Color Direction

Recommended base palette:

- Warm cream: `#F7F1E8`
- Charcoal: `#1F2430`
- Coral: `#C85C3A`
- Buy green: `#2E7D5B`
- Bye rust: `#B4493D`

Optional accent palette:

- Soft gold: `#B7871B`
- Sand: `#E7D6BF`

## Typography Direction

Suggested tone:

- clean sans-serif wordmark
- modern geometric shapes
- strong readability
- slightly premium, not luxury

Good style references:

- bold neo-grotesk
- rounded geometric sans
- editorial serif + modern sans combination

## Symbols To Explore

Use one or more of these ideas:

- split wordmark between "Buy" and "Bye"
- checkmark vs x mark
- price tag with split approval/rejection
- shopping cart with subtle decision mark
- mirrored `B / B` monogram
- directional fork or divider line
- thumbs up vs thumbs down abstraction

## What To Ask Gemini For

Use prompts that request:

- vector logo style
- flat branding presentation
- clean white or cream background
- strong readability
- app icon plus wordmark
- multiple directions in one batch

Also ask Gemini to avoid:

- mockup-heavy presentation
- 3D glossy effects
- cluttered icons
- too many gradients
- childish illustrations

## Prompt Set 1: Split Wordmark

**Prompt**

```text
Design a modern vector logo for a product comparison app called "BuyOrBye". The brand helps users compare products by price, quality, and value before buying. Create a clean split wordmark where "Buy" feels approved and confident, and "Bye" feels like a rejected or skipped option. Use a smart, modern, trustworthy visual style. Include a subtle decision element such as a divider, checkmark, or directional split inside the lettering. Use a warm cream background with charcoal, coral, and green accents. Make it look like a polished tech startup brand. Show 4 distinct logo variations, flat presentation, no mockups, no 3D.
```

**Best for**

- primary brand logo
- website header logo
- clean startup identity

## Prompt Set 2: Price Tag Icon

**Prompt**

```text
Create a clean vector logo for "BuyOrBye", a web app that compares products and helps users decide what is worth buying. Explore a logo concept using a price tag icon split into two states: approved vs rejected, buy vs bye. Keep the icon simple, scalable, and suitable for app icon use. Pair it with a modern wordmark. Use a warm neutral background with charcoal, coral, green, and muted rust tones. The logo should feel smart, decisive, minimal, and premium. Show several variations in a flat brand board, no mockups, no photorealism.
```

**Best for**

- app icon direction
- favicon exploration
- decision-based symbol system

## Prompt Set 3: Monogram

**Prompt**

```text
Design a monogram logo for "BuyOrBye" using the letters B and B. The logo should represent product comparison, approval vs rejection, and smart buying decisions. Explore mirrored, split, or symmetrical monogram concepts inside a rounded square or circular icon. Pair the symbol with a clean modern wordmark. Style should be modern SaaS branding, minimal, memorable, and scalable. Use charcoal, green, coral, and warm cream. Present multiple vector options on a simple background, no 3D, no merchandise mockups.
```

**Best for**

- mobile app icon
- social avatar
- compact brand mark

## Prompt Set 4: Premium Editorial

**Prompt**

```text
Create a premium editorial-style logo for a comparison app named "BuyOrBye". The brand should feel intelligent, clear, modern, and trustworthy. Combine a refined serif or semi-serif feeling with a modern sans-serif structure. The logo should suggest thoughtful decisions, comparing options, and avoiding buyer regret. Keep the icon subtle and sophisticated, possibly using a split line, checkmark, or tag motif. Use a warm cream background with charcoal and coral as primary tones, with green used only as a controlled accent. Generate 4 polished vector directions, flat layout, minimal presentation.
```

**Best for**

- higher-end brand feel
- elegant homepage branding
- more distinctive design language

## Prompt Set 5: Bold Startup

**Prompt**

```text
Generate bold startup-style logo concepts for "BuyOrBye", a smart shopping decision app. The logo should feel fast, confident, clear, and digitally native. Focus on strong shapes, a readable wordmark, and a simple icon that communicates buy vs skip. Explore checkmark and x symbolism in an abstract way, not too literal. Use clean vector design, flat colors, warm cream background, charcoal text, coral primary accent, and green for approval. Show 6 distinct concepts with app icon and wordmark variations. No gradients unless very subtle, no shadows, no mockups.
```

**Best for**

- landing page branding
- startup identity exploration
- high-clarity options

## Prompt Set 6: App Icon Only

**Prompt**

```text
Design app icon concepts for a product comparison app called "BuyOrBye". The icon should communicate price comparison, decision-making, and buy vs reject in a simple scalable symbol. Explore ideas like a split price tag, checkmark vs x, directional fork, or double B monogram. Keep the style minimal, flat, modern, and instantly recognizable at small sizes. Use charcoal, green, coral, and warm cream. Show multiple icon-only variations in a flat grid on a neutral background.
```

**Best for**

- mobile icon only
- favicon
- UI icon system

## Negative Prompt Add-On

Append this if Gemini starts generating overly flashy results:

```text
Avoid 3D effects, glossy gradients, mascots, cartoon style, ecommerce clichés, shopping bags with too much detail, crowded layouts, tiny unreadable text, and generic marketplace branding.
```

## Shortlist Recommendation

If you want the strongest first round, start with these 3:

1. Split Wordmark
2. Price Tag Icon
3. Monogram

That mix usually gives:

- one clean primary logo
- one symbol-first app icon idea
- one compact premium mark

## What To Look For In Results

When comparing outputs, favor the directions that have:

- strong readability
- easy recognition at small size
- clear contrast between "Buy" and "Bye"
- a symbol that still works without the full wordmark
- a look that feels like a trusted tool, not a discount store

## Best One-Line Prompt

If you only want one prompt to start with, use this:

```text
Design a modern vector logo for "BuyOrBye", a web app that helps users compare products by price, quality, and value before buying. The logo should feel smart, decisive, trustworthy, and modern, with a clean wordmark and a simple icon that represents buy vs skip. Explore split wordmark, price tag, and monogram directions using charcoal, coral, green, and warm cream. Flat brand presentation only, no mockups, no 3D.
```
